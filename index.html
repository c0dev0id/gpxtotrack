gpxtotrack

Convert a Garmin GPX route file into an enriched GPX with a clean route (densified from Garmin's shaping points), a dense track, and the waypoints — stored in generic GPX 1.1 wherever possible.

Drop GPX files here, or

Route color

GPX 1.1 has no color field. Keeping it requires a Garmin extension in the output.

Routing waypoint types

Garmin marks route points as via-points or shaping points. Only useful on Garmin routing engines; no GPX 1.1 equivalent. Removing gives a cleaner file.

Third-party extensions

Extensions from non-Garmin tools (e.g. Rümo routing profile). Cannot be expressed in GPX 1.1.
